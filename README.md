# Sypnose

**AI Orchestration Methodology** - 42 Models, 29 Agents, 1 Method.

Sypnose is a methodology where multiple AI agents work as an organized team, coordinated by a Service Manager, communicating via Knowledge Hub.

## What is Sypnose?

A neologism combining:
- **Synapse** — neural connections between agents
- **Synopsis** — global vision, unified dashboard
- **Hypnosis** — deep focus, each agent locked on its task

**Intelligent connections that deliver a unified picture of something complex.**

## The 6-Tag Protocol

Every task must pass 6 mandatory gates:

1. **PLAN** — What are we going to do?
2. **TASK** — What exactly needs to happen?
3. **MODEL** — Which AI model handles this?
4. **BORIS** — Is there a safety net?
5. **VERIFICATION** — How do we prove it works?
6. **EVIDENCE** — Show me the proof.
